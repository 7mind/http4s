Community

Getting Help

IRC

The http4s team and many of its users are available to chat at #http4s on Freenode. The channel is not logged1. Come join the discussion, ask for help, or just lurk!

Mailing list

We've started a [Google Group](https://groups.google.com/group/http4s). This lacks the real time give-and-take of IRC, but you won't netsplit before you get a reply. IRC has been our more popular support channel, but we're here to help. Don't be shy!

GitHub Issues

http4s uses GitHub issues. If you think you've hit a bug or have an idea of what we should do next, please don't hesitate to submit an issue. Your bug reports and request make the project better for everyone.

Contributing

http4s is an open source project, and depends on its users to continue to improve. Here's how you can help:

  • Submit an issue. Bug reports, feature requests, whatever. No nit too small to pick.
  • Send us a pull request with your original work, compatible with the project's Apache 2 Licence. Before a major change, please file an issue or contact us first, so we're not duplicating effort.
  • If you found the project helpful, pay it forward with a contribution to the docs. This site is a simple Jekyll project in our main repo.
  • We're @http4s on Twitter. Tell your friends.
